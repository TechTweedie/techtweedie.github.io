Generate Power Platform Solution Documentation with Azure DevOps

  • Power Platform
  • Azure DevOps
  • Documentation
  • DevOps
  • Community Call

Watch the Demo

About

This demo shows a DevOps-based approach to generating Power Platform documentation automatically from solutions stored in Azure DevOps. It demonstrates how solution metadata can be converted into Markdown, diagrams, and Word documents as part of a pipeline, removing the need for manual documentation.

This demo was presented at the Power Platform Monthly Community Call on the 19th of November 2025.

Everyone is welcome to these calls — download recurrent invites from https://aka.ms/community/calls.
